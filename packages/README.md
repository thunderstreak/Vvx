# Vvx

> Vvx UI 一套基于Vue2.0开发的一套简单常用的移动端UI组件。

## Install

```shell
npm install vvxui
```
## Quick Start
``` javascript
import Vue from 'vue'
import Vvx from 'vvxui'

Vue.use(Vvx)

// or
import {
  	Dialog,
  	Loding
  	// ...
} from 'Vvx'

Vue.component(Dialog.name, Dialog)
Vue.component(Loding.name, Loding)
```

## Browser Support
* android4.4+
* ios9+

## Commonly used Components
- Dialog
- Loding
- PickerAddress
- PickerTime
- PickerSelect
- Swiper
- Switch
- More in development...
